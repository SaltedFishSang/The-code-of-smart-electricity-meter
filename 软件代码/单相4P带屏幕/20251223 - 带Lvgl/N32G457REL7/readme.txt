1、功能说明

    1、此例程为无操作系统（#define NO_SYS 1）http网络测试例程，用开发板当做简单的网页服务器，通过浏览器访问可现实简单网页

2、使用环境

    /* 软件开发环境：当前工程使用的软件工具名称及版本号 */
        IDE工具：KEIL MDK-ARM 5.30

        /* 硬件环境：工程对应的开发硬件平台 */
        开发板：N32G457QE_EVB V1.1

3、使用说明

    /* 描述相关模块配置方法；例如:时钟，I/O等 */
        SystemClock：100MHz


    /* 描述Demo的测试步骤和现象 */
    1.开发板PA9,PA10作为串口输出，串口工具设置波特率为115200
    2.将电脑主机和开发板连接在同一网段内。
    3.编译后下载程序复位运行。
    4.串口工具将会打印信息，包括开发板被分配到的IP地址（如果使用DHCP，反之打印默认IP）。
    5.在电脑主机打开浏览器，直接输入ip地址。
    6.浏览器将会显示简单的网页信息。

4、注意事项
    1.先插入网线，再将开发板复位运行。首次上电建议先插网线。
    2.电脑主机和开发板在同一网段。
    3.注意J16、J17连接两个跳线帽
